## Getting started

### Installing Sphinx

Before you can build the documentation, you'll need to have Sphinx on you
system. We suggest that you do it witihin a virtualenv, like so:
```
$ virtualenv -p python3 $HOME/py3
$ $HOME/py3/activate
$ which pip
/$HOME/py3/bin/pip
$ pip install Sphinx
```

### Create html pages

```
$ make html
```
